Lexicon JS interface front-end roadmap
=======================================

Общее:
* все выводимые строки хранятся в messages.csv.
* где нужен простейший вкладочный интерфейс - на css может быть как на w3schools
ἄρα слова из трех букв смотреть
* поиск по буквам без диакритики, чтобы облеч. ударение не мешало найти лимму с острым
* σσ на ττ

проблемы
=========
τέμνω не выводит значения.
λεξι, ἐφύ - в минингс проблемы в данных

Страница слова
=============
* поле для ввода другого слова, как во всплыв. окне

* чтобы тоже двойной щелчок работал
* галочка "Грамматические термины на латинском"


описание: класс глагола. и 6 представительных форм.
если причастие, то form причастие и далее описание

словарная статья: вкладочки, выбор на каком языке статья.
скан страницы Вейсмана, Дворецкаго, Лидделл-Скотта с нужным словом

таблица форм:
* сортировка в таблице
* вкладочки - времена, залоги, наклонения.
* настраиваемая - какие столбцы, строки. подтаблицы.

примеры употребления:
* всех форм слова
* из закладок пользователя
* из поиска в интернете, для чего web-scraping

============
* чтобы английские слова тоже мог смотреть, на etymonline
* контекстное меню в контент-скрипте.
позволяющее посмотреть слова, начинающиеся/окачивающиеся на первые/последние 3-5 букв данного слова
* замена на правильный знак ударения. τὦ например неправильно заменяет. йоту подписную портит
* install right-click menu "turn monotonic".
